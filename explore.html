Learn HTML and CSS

EXAMPLE CODE:
HTML (HyperText Markup Language) and CSS (Cascading Style Sheets) are the building blocks of web design. Below is an example of basic HTML and CSS:

    <!DOCTYPE html>
    <html>
      <head>
        <style>
          body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
          }
          h1 {
            color: blue;
          }
        </style>
      </head>
      <body>
        <h1>Hello, World!</h1>
        <p>Welcome to learning HTML and CSS.</p>
      </body>
    </html>

You can use this structure to build simple web pages. Click "Run Code" to practice coding.

HTML Elements

  • <li> - List Item
  • <ul> - Unordered List
  • <a> - Anchor (Link)
  • <span> - Inline Container
  • <p> - Paragraph
  • <h1> to <h6> - Headings
  • <div> - Division
  • <nav> - Navigation
  • <link> - External Resource Link

CSS Properties

  • color - Text Color
  • background-color - Background Color
  • background-image - Background Image
  • transform - Transform Element
  • display - Element Display
  • margin - Space Outside Element
  • gap - Gap Between Elements
  • @keyframes - Define Animations
  • animation - Apply Animation
  • border - Element Border
  • border-radius - Rounded Corners
  • flex-direction - Flexbox Direction
  • transition - Smooth Changes
  • text-decoration - Text Styling
  • box-shadow - Shadow Around Box
  • text-shadow - Shadow Around Text

These elements and properties are the foundation for creating and styling web pages. Experiment with them in the practice section.
